Help and Support

Everything you need to use InsightKPI effectively

Getting Started

  • Go to Add KPI to track a new business metric
  • Enter a value, date, and optional target
  • View all KPIs in your dashboard

Dashboard Features

  • View KPI performance trends
  • Compare two KPIs side-by-side
  • Filter KPIs by date range
  • Identify best and worst performing KPIs
  • Status indicators:
    • Green = On target
    • Amber = Close to target
    • Red = Needs attention

Managing KPIs

  • Add new KPIs using the Add KPI button
  • Edit or delete KPIs from the dashboard menu
  • Click on a KPI to view full details

Exporting Data

  • Download your KPI data as a CSV file
  • Generate a PDF performance report
  • Use exports for reports or presentations

Account Management

  • Update your password in the Profile page
  • Admins can manage users and activity logs
