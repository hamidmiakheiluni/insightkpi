Quick guide to using InsightKPI.

Getting started

  • Register an account, then log in.
  • Go to Add KPI to enter a metric (e.g., Sales, Profit, Customers).
  • Return to the dashboard to view the table and chart.

Filtering & analysis

  • Use the KPI dropdown to view one KPI at a time.
  • Use Start/End dates to analyse performance over a time period.
  • The chart and table update based on your filters.

Reports

  • Export PDF downloads a printable report.
  • Export CSV downloads raw data for Excel.

Edit/Delete

  • Use Edit to update an entry.
  • Use Delete to remove an entry.
  • You can only edit/delete your own KPI entries.
